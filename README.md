# BambouCamer

Site multipages moderne pour BambouCamer, construit avec **Next.js**, **React**, **Tailwind CSS** et **Framer Motion**.

## Démarrage

```bash
npm install
npm run dev
```

Ouvrez [http://localhost:3000](http://localhost:3000).

## Scripts

- `npm run dev` — serveur de développement
- `npm run build` — build de production
- `npm start` — servir le build
